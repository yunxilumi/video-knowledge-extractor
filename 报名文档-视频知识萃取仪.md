## 1. 创意名称 + 创意介绍

**创意名称**：视频知识萃取仪（Video Knowledge Extractor）

**想解决什么问题**：
在 B 站和小红书上刷了大量"干货视频"——收藏了上百个视频，却再也没打开过；看完觉得"学到了"，三天后什么都记不起来；想回头找某个知识点，要在海量收藏里翻半天。视频是线性播放的，但知识是结构化的——二者天然不匹配。市场上缺少一个工具，能把视频内容"拆解"成真正可回顾、可消化、可分享的知识形态。

**为什么会想到做这个**：
我自己就是重度 B 站/小红书学习者，收藏夹里躺着几十个"必看教程"，但大部分从未被打开第二次。我意识到问题不在于内容不够好，而在于**没有一种方式让视频里的知识真正留存下来**。现有的收藏功能只是"打了个标签"，没有帮用户完成"信息→知识"的转化。AI 的文本理解与概括能力在 2025-2026 年已经足够成熟，但市场上仍然没有一个产品专门做"视频→结构化知识"的转化。

**大概是什么产品**：
Web 应用。用户粘贴 B 站/小红书视频链接，产品自动获取字幕，通过 AI 将视频内容梳理成结构化的文字笔记（大纲 + 重点摘要 + 时间戳引用）。用户还可以选择开启"可视化"——一键生成知识卡片图片或思维导图。此外，基于用户历史链接的内容主题分析，产品可以提供轻量的个性化视频推荐（用户可选关闭）。核心流程是：链接 → 理解 → 结构化 → 输出。

## 2. 目标用户及痛点

**面向哪些用户**：
- **🎓 备考学生**：在 B 站看考研/考证/留学教程，需要高效提炼重点、生成复习笔记
- **💼 职场学习者**：在小红书/B 站学技能（PPT、Excel、编程、设计等），看完了但记不住
- **✍️ 内容创作者**：需要参考竞品视频结构，快速分析优质视频的内容框架
- **📚 知识收藏爱好者**：收藏了一大堆"干货视频"，但从未回顾，收藏夹成了"知识的坟墓"

**使用场景**：
- 📖 **备考复习**：看了一小时"考研政治思维导图"视频 → 粘贴链接 → 产品输出 5 个章节、22 个考点 → 一键生成思维导图知识卡片 → 保存复习
- 🔍 **快速学习**：刷到一个 3 分钟的"如何做 PPT"视频 → 粘贴链接 → 产品输出 3 个技巧、6 个案例的文字笔记 → 直接消化核心内容
- 📐 **创作参考**：看到同行爆款视频 → 粘贴链接 → 产品输出视频结构分析（开头钩子→痛点→解决方案→案例→CTA）→ 作为创作模板参考
- 📲 **分享知识**：看完一个高质量教程 → 生成精美知识卡片 → 分享到朋友圈/小红书

**当前痛点**：
1. 收藏功能只是"存了个位置"，没有帮用户做任何知识转化
2. 视频是线性播放，知识点需要自己手动整理，耗时费力
3. 看完就忘，时间花了但知识没留下
4. 想分享学到的内容，但不知道怎么组织成好看的格式
5. 想找同类内容，但平台算法推荐不够精准，每个平台各推各的

## 3. 价值与意义

**效率价值**：
把"看完视频→手动记笔记→整理归纳"这个过程从 30-60 分钟缩短到 1 分钟。AI 自动完成分段、提炼、结构化，用户只需确认和消化。默认输出结构化文字笔记，让信息转化为可回顾的知识形态。

**学习价值**：
"收藏"不等于"学会"。产品帮用户完成了学习的**内化环节**——将线性的视频叙事转化为层级分明的结构化知识，用户可以随时回顾、搜索、关联，真正把视频里的知识变成自己的。

**用户主权价值**：
用户可以选择**是否开启可视化**（默认仅输出文字笔记），可以选择**是否开启推荐**（默认不推荐）。产品定位是"用户的个人知识转化工具"，而不是另一个推荐引擎。所有内容主动权都在用户手中。

**创作者价值**：
为内容创作者提供了一个"视频结构解构工具"，可以快速分析优质内容的框架逻辑，辅助创作。
